for语句:

            原形：  {for:var max min}
                    循环条件满足时运行。
                    {else}
                    循环条件不满足时运行。
                    {/for}
            var：   循环变量名
            max：   循环的最大值（整数），如果max小于min，则将会使用倒序循环。
            min：   可选，默认为0。循环的最小值（整数）。
